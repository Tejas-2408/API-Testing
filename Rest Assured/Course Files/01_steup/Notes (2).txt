Rest Assured
-------------

It is an open-source Java library and is free to use.
It is used for validating and testing Java APIs.
supports all HTTP methods like GET, POST, PUT, DELETE etc.

We can integrate Rest Assured with all major automation frameworks 
like TestNG, JUnit as well as integrate it with maven and CI/CD.

Pre-requisites
------------

1)  Eclipse IDE/Java 11+
2) TestNG
